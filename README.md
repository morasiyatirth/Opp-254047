**Submitted to:** Prof. Mayank Pathak

# 1. Introduction
This is a console-based application developed in C++ to manage library records efficiently. It allows users to add new books to a digital database and view the existing collection.

# 2. Objectives of the Project
* To implement Object-Oriented Programming (OOP) concepts.
* To practice data management using C++ Vectors.
* To create a functional, user-friendly Command Line Interface (CLI).

# 3. Methodology / System Design
The system uses a `Book` class to encapsulate data (Book ID and Name). It utilizes the `std::vector` container for dynamic memory management, allowing the library to grow as more books are added.

# 4. Implementation / Source Code
The core logic is contained in `main.cpp`, featuring a menu-driven interface using `while` loops and `switch-case` statements.

# 5. Conclusion
This project successfully demonstrates the use of classes and objects in C++. It highlights how encapsulation helps in keeping data and functions bundled together for better code organization.
